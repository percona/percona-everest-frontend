---
description: A component to use on composition files that enables theme mode toggling with Everest theme.
---

import { EverestCompositionViewer } from './everest-composition-viewer';

This component is a composition of `CompositionViewer` with `theme={EverestTheme.getTheme}`.

### Component usage

```js
<EverestCompositionViewer>
  <p>Hello World</p>
</EverestCompositionViewer>
```
