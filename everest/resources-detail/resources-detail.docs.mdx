import { ResourcesDetail } from './resources-detail';

This component is composition of `ProgressBar` and `Input` componets. For displaying resources.

### Component usage

```js
<ResourcesDetail
  value={1}
  total={10}
  inputValue={inputValue}
  setInputValue={setInputValue}
  label="CPU"
  labelProgressBar="Using 112.52 CPU (16.7%) of 675.33 CPU in total"
  units="CPU"
/>
```
