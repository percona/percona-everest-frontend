---
description: A custom DialogTitle component.
---

This DialogTitle adds an optional close icon.

### Component usage
```js
<Dialog open={open}>
  <DialogTitle onClose={() => setOpen(false)}>
    Modal title
  </DialogTitle>
  <DialogContent dividers>
    <Typography>
      Lorem ipsum
    </Typography>
  </DialogContent>
  <DialogActions>
    <Button autoFocus variant="outlined" onClick={() => setOpen(false)}>
      Cancel
    </Button>
    <Button variant="contained">
      Accept
    </Button>
  </DialogActions>
</Dialog>
```
