---
description: AutoCompleteInput component.
---

A wrapper around MUI's AutoComplete and RHF's controller. MUI's TextField is used for renderInput method.

### Component usage
```js
<AutoCompleteInput
  name={DbWizardFormFields.storageLocation}
  control={control}
  label={Messages.storageLocation}
  loading={isFetching}
  options={backupStorages}
  autoCompleteProps={{
    isOptionEqualToValue: (option, value) => option.id === value.id,
    getOptionLabel: (option) => typeof option === 'string' ? option : option.name,
  }}
/>
```
