---
description: A component that allows dynamic addition / removal of text input fields.
---

### Component usage

```js
const methods = useForm({
  defaultValues: {
    people: [
      {
        person: 'John',
      },
    ],
  },
});

<FormProvider {...methods}>
  <TextArray fieldName="people" fieldKey="person" placeholder="Person" />
</FormProvider>;
```
