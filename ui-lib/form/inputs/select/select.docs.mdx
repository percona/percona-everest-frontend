A wrapper around RHF Controller and MUI Select with an optional label

### Component usage

```js
const { control } = useFormContext();

<SelectInput name="company" label="Company" control={control}>
  <MenuItem value="Percona">Percona</MenuItem>
  <MenuItem value="Percona">Other</MenuItem>
</SelectInput>;
```
